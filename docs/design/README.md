# Design References

Use this folder for UX references, screenshots, sketches, and visual inspiration that inform product direction but are not application data.

Recommended structure:

- `ux-references/`: screenshots or images used as UX inspiration.
- `notes.md`: short notes explaining what each reference is meant to inspire.

Guidelines:

- Only include images you have the right to store in the repository.
- Prefer screenshots you created yourself, public-domain images, or links with notes instead of copying copyrighted images.
- Do not place proprietary Amazon screenshots, internal mocks, or confidential assets here.
- If an image should be processed by the chatbot as user-provided input, that belongs in the app upload flow later, not in this design reference folder.
